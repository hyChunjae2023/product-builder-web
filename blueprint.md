# Lotto Number Generator

## Overview

A simple web application to generate random lottery numbers.

## Features

*   Generates 6 unique random numbers between 1 and 45.
*   Displays the numbers in a visually appealing way using Web Components.
*   Modern and clean user interface.

## Current Plan

*   **index.html:**
    *   Update the title to "Lotto Number Generator".
    *   Create the main structure of the application with a title, a button to generate numbers, and a container to display the numbers.
*   **style.css:**
    *   Add styles for the overall layout, button, and the lottery number display.
    *   Create a modern and visually appealing design.
*   **main.js:**
    *   Implement the lottery number generation logic.
    *   Create a Web Component (`lotto-ball`) to display each number.
    *   Add an event listener to the button to trigger the generation and display of the numbers.
